# Catálogo de Filmes

Projeto desenvolvido como teste técnico, com o objetivo de criar uma aplicação de catálogo de filmes, consumindo a API do TMDB, permitindo a busca de filmes e o gerenciamento de uma lista de favoritos.

O projeto foi estruturado com backend e frontend separados, mantendo responsabilidades bem definidas entre API e interface.

## Tecnologias utilizadas

### Backend
- PHP
- Laravel
- SQLite (default)
- MySQL (opcional)
- Integração com API externa (TMDB)

### Frontend
- Vue.js 3
- JavaScript
- Vite
- Axios
- Vue Router
- Pinia

## Estrutura geral do projeto
```
catalogo-filmes/
├── backend/
└── frontend/
```
## Objetivo do projeto

- Consumir dados da API do TMDB
- Permitir busca de filmes
- Exibir informações básicas dos filmes
- Gerenciar uma lista de filmes favoritos
- Demonstrar separação entre frontend (SPA) e backend (API REST)
